# CSS

## Utilizing CSS

```
<!-- Inline -->
<h1 style="color: blue;">Hello</h1>

<!-- Embedded -->
<style>
h1 {
    color: blue;
}
</style>

<!-- Export -->
<link rel="stylesheet" href="style.css" />

<!-- In style.css -->
h1 {
    color: blue;
}
```
